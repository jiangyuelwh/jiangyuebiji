🧠 AI 编程学习大纲（零基础 · 实战导向）

核心理念： 先学会用 AI 造轮子，再理解轮子怎么造。
学习周期： 约 6-8 周，每天 1-2 小时。

📅 第一阶段：环境搭建与工具链（第 1 周）

目标： 搭好 AI 编程的工作流，别在安装软件上卡住。

1.1 安装核心工具

  • Python 环境： 安装 Anaconda 或 Miniconda（省去环境配置烦恼）
  • 代码编辑器： 安装 VS Code，并熟悉基本操作
  • AI 编程助手（任选其一）：
    • 国内推荐：通义灵码（阿里云，VS Code 插件）、文心快码
    • 国际推荐：GitHub Copilot、Cursor 编辑器
  • 虚拟环境： 学会 conda create 和 pip install，养成隔离环境的习惯

1.2 必学 Python 基础速览

  • 变量、数据类型、条件判断、循环
  • 列表与字典的核心操作
  • 函数定义与调用
  • 关键技能： 学会在 VS Code 里用 AI 助手自动补全和解释代码

✅ 阶段作业

  • 用 AI 助手在 VS Code 里写一个“学生成绩管理”小脚本，能增删查改。

📅 第二阶段：开启 AI 对话编程（第 2 周）

目标： 掌握与 AI 协作写代码的语法——提示词工程。

2.1 提示词框架

  • 角色扮演： “你是一名资深 Python 工程师……”
  • 分步拆解： 学会把大问题拆成 AI 能一口吃下的小问题
  • 喂示例（Few-Shot）： 给 AI 一两个例子，让它模仿
  • 框架推荐： CRISPE 框架（角色-背景-意图-风格-期望）

2.2 实战演练

  • 用对话让 AI 帮你写爬虫（如抓取天气）
  • 让 AI 帮你写 Python 小游戏（贪吃蛇、2048）
  • 核心技巧： 报错时，直接把错误信息丢给 AI，学习它如何修复

✅ 阶段作业

  • 全程不手写代码，用对话让 AI 生成一个带界面的“待办事项”小工具。

📅 第三阶段：机器学习速通（第 3-5 周）

目标： 理解 AI 的底层逻辑，学会用最经典的库。

3.1 理论直觉课

  • 机器学习是什么： 从数据中找规律
  • 三大任务： 分类（找猫狗）、回归（测房价）、聚类（分用户群）
  • 炼金术心态： 数据是原料，模型是丹炉，调参是火候——先别钻数学公式！

3.2 核心库入门

  • NumPy： 数据计算的基石（理解数组）
  • Pandas： 清洗表格数据的神器（读 CSV、处理缺失值）
  • Matplotlib/Seaborn： 画图看趋势（直方图、散点图）

3.3 经典模型实战

  • Scikit-learn： 用 5 行代码跑起线性回归、决策树
  • 完整流程： 加载数据 → 清洗 → 切分训练/测试集 → 训练 → 评估准确率
  • 跟着做： Kaggle 的“泰坦尼克号幸存者预测”项目——AI 小白必刷课

✅ 阶段作业

  • 找一个你感兴趣的 Excel/CSV 数据集，用 Sklearn 建模预测，并画图展示结果。

📅 第四阶段：深度学习入门（第 6-7 周）

目标： 用最友好的方式，碰一碰神经网络。

4.1 框架选择

  • 新手首选 PyTorch（比 TensorFlow 更 Pythonic，社区活跃）
  • 了解张量是什么、GPU 加速是咋回事

4.2 图像识别入门

  • hello world 项目： MNIST 手写数字识别（AI 界的“新手村”）
  • 必学概念： 卷积神经网络直觉理解（扫描、提取特征）
  • 进阶可选： 用预训练模型 ResNet 做图片分类（站在巨人肩膀上）

4.3 免费算力

  • 注册 Kaggle Notebook 或 Google Colab，用云 GPU 跑模型

✅ 阶段作业

  • 用 PyTorch 在 Colab 上训练一个 MNIST 模型，识别自己手写的数字照片。

📅 第五阶段：调用大语言模型 API（第 8 周）

目标： 开发真正的 AI 应用，接触 RAG 和智能体。

5.1 API 调用

  • 获取 DeepSeek / 通义千问 / OpenAI 的 API Key
  • Python 发送请求，拿到回复
  • 项目： 做一个命令行“AI 翻译官”

5.2 进阶应用（概念了解）

  • RAG（检索增强生成）： 让 AI 读你的本地文件再回答（个人知识库）
  • Agent（智能体）： 让 AI 能自动调用搜索、计算器等工具
  • LangChain / LlamaIndex： 了解这两个主流框架的名字和用途

✅ 阶段作业

  • 做一个“AI 每日新闻简报生成器”，调用模型总结指定话题的最新信息。

🧰 小白救急指南（遇到问题怎么办）

  1. 先问 AI： 把报错原样丢给 ChatGPT/Kimi/DeepSeek，90% 能解决。
  2. 再查文档： 官网 Tutorial 永远是第一手资料。
  3. GitHub Issues： 搜错误关键词，常有人踩过坑。
  4. 心态管理： 环境报错比算法还玄学，卡住 3 天很正常，先跳过也行。
